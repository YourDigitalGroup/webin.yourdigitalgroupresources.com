Bonus scoring

Play it your way.

Turn on bonus points for a gross birdie, eagle, or a hole-in-one. Add the skunk rule — win a hole by two clear strokes and take every point on it. Split the round into 6-, 9-, or 18-hole matches. All optional, all set at the tee.

How it's scored

3 steps
01

Set up

Pick a game and match length, add players, flip on any bonus points. You get a room code to share.

02

Score

One scorekeeper enters gross strokes per hole. Points split automatically — ties, skunks, and bonuses handled.

03

Watch

Everyone else opens the code and watches the leaderboard update live, no refresh, no waiting for the turn.
